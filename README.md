The following is a game with a bitmap object the player controls made of 4 primitives: rectangle, triangle, ellipse, circle.
The goal is to use the object to move around the screen with the arrow keys collecting bullets.
The bullets are another 16 x 16 bitmap made of two primitives.
There is a timer for 30 seconds, which the player spends collecting bullets.
The score is also kept track of in the bottom right corner.
When the timer hits 0, the program closes.
Allegro is needed to play.
